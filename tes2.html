NURMA's PROFILE

LAPORAN PRAKTIKUM APLIKASI KOMPUTER

Hardware

hardware adalah segala piranti atau komponen dari sebuah komputer yang sifatnya bisa dilihat secara kasat mata dan bisa diraba secara langsung. Dengann kata lain hardware merupakan komponen yang memiliki bentuk nyata.

Sistem Operasi Windows

Windows merupakan sebuah Sistem Operasi hasil pengembangan dari Sistem Operasi DOS, karena kemudahan dari Sistem Operasi ini menjadikan Ms. Windows banyak digunakan oleh pengguna diseluruh dunia.

Sistem Operasi Linux

Linux merupakan suatu perangkat lunak atau software sistem operasi yang sifatnya Open source. Dengan Linux maka pengguna dapat memperoleh software yang lengkap dengan source code-nya bahkan memodifikasinya dengan dibawahi lisensi GNU.

Microsoft Word

Microsoft Word merupakan perangkat lunak yang biasa digunakan untuk mengolah kata meliputi membuat, mengedit, dan memformat Selengkapnya. Perangkat lunak pengolah kata atau word processing adalah program yang digunakan untuk mengolah Selengkapnya berupa yang berkaitan dengan tulis menulis.

Microsoft Excel & PPT

Microsoft Excel atau Microsoft Office Excel adalah sebuah program aplikasi desktop yang berfungsi sebagai lembar kerja spreadsheet. Sedangkan Microsoft PowerPoint adalah software yang dipakai untuk merancang bahan presentasi dalam bentuk slide.

Desain Grafis

Design Grafis adalah proses komunikasi menggunakan elemen visual, seperti tulisan, bentuk, dan gambar yang dimaksudkan untuk menciptakan persepsi akan suatu pesan yang disampaikan. Adapun bentuk-bentuk desain grafis adalah ilustrasi, tipografi, fotografi, atau grafis motion.

Multimedia

Pengertian “multimedia” menurut Hackbarth (1996: 229) adalah: Multimedia diartikan sebagai suatu penggunaan gabungan beberapa media dalam menyampaikan informasi yang berupa teks, grafik atau animasi grafis, movie, video, dan audio. Multimedia meliputi hypermedia dan hypertext.

HTML & CSS

Hypertext Markup Language (HTML) adalah suatu metode untuk mengimplementasikan konsep hypertext dalam suatu naskah. Sedangkan CSS (Cascading Style Sheets) adalah kumpulan perintah yang digunakan untuk menjelaskan tampilan sebuah halaman situs web dalam mark-up language.

Pemrograman Web (Hosting)

Hosting adalah suatu layanan yang berfungsi sebagai tempat untuk menyimpan semua data (database) sebuah website sehingga dapat diakses melalui internet. Dalam hal ini data tersebut dapat berupa Selengkapnya, gambar, video, email, aplikasi, dan lain sebagainya.

Augmented Reality

Augmented Reality adalah penggabungan antara dunia nyata dan dunia maya, di mana objek virtual overlayed pada dunia nyata. Dari segi teknis, teknologi Augmented Reality merupakan teknologi transformatif, dimana sistem interaksi melingkupi keseluruhan lingkungan di luar tampilan layar.
